# Antigravity CLI Custom Instructions

These instructions are loaded by the AI coding assistant for all sessions within this workspace.

## Terminal-Friendly Output Rules
- **No LaTeX Formatting:** Never use LaTeX formatting block delimiters (`$$...$$`) or inline delimiters (`$ ... $`). These fail to render correctly in terminal environments.
- **Formulas:** Always represent math equations, formulas, and expressions using standard code blocks (e.g., `text` or `c`) or plain ASCII text (e.g., `Feedforward_Output = (Kv * Target_Velocity) + (Ka * Target_Acceleration) + (Ks * direction)`).
- **Bullet Points:** Use standard Markdown hyphens (`-` or `*`) instead of special characters like `•`.
- **Special Symbols:** Use standard words or plain ASCII equivalents (e.g., `degrees`, `delta`, `alpha`, `->`) instead of special symbols (`°`, `Δ`, `α`, `→`).
